## 5.1 原子的核模型

### 5.1.1 原子的结构

- 原子由 **原子核（nucleus）** 和核外 **电子（electrons）** 组成。
- 原子核带正电，体积小但质量集中；电子带负电，质量极轻。
- **离子形成**：失去电子 → 正离子；得到电子 → 负离子。

#### α 粒子散射实验（金箔实验）
- **现象**：大部分 α 粒子直穿，少数小角度偏转，极少数反弹。
- **结论**：原子大部分是空的；正电荷和质量集中于极小原子核；原子核带正电。

### 5.1.2 原子核的组成

| 粒子 | 电荷 | 相对质量 | 位置 |
|------|------|----------|------|
| 质子 | +1 | 1 | 核内 |
| 中子 | 0 | 1 | 核内 |
| 电子 | –1 | 1/1836 | 核外 |

- **质子数 Z** = 质子数目，决定元素种类。  
- **核子数 A** = 质子数 + 中子数。中子数 = A – Z。
- **核素符号**：写成 `^A_Z X`，例如 `^12_6 C`（碳-12），`^235_92 U`（铀-235）。
- **同位素**：质子数相同、中子数不同的原子。化学性质相同，物理性质不同。
- **核裂变**：重核分裂，释放能量和中子。示例：  
  `^235_92 U + ^1_0 n → ^141_56 Ba + ^92_36 Kr + 3 ^1_0 n` + 能量
- **核聚变**：轻核结合，释放巨大能量。示例：  
  `^2_1 H + ^3_1 H → ^4_2 He + ^1_0 n` + 能量

## 5.2 放射性

### 5.2.1 放射性的探测

#### 背景辐射（background radiation）
- 环境中始终存在微弱电离辐射。主要来源：
  1. 氡气（从岩石土壤释放）
  2. 岩石和建筑物（如花岗岩）
  3. 食物和饮料（如香蕉含钾-40）
  4. 宇宙射线

#### 探测核辐射
- 常用仪器：盖革-米勒计数器（GM tube）与计数器连接。
- **计数率**：单位时间的计数，单位 counts/s 或 counts/min。
- **校正计数率** = 实测计数率 – 本底计数率。

### 5.2.2 三种核辐射

| 类型 | 本质 | 电离能力 | 穿透能力 | 电场/磁场偏转 | 速度（约） |
|------|------|----------|----------|----------------|------------|
| α | 氦核（^4_2 He） | 最强 | 最弱（一张纸） | 偏向负极，偏转小 | 0.1c |
| β | 高速电子（^0_-1 e） | 中等 | 中等（几 mm 铝） | 偏向正极，偏转大 | 0.9c |
| γ | 高频电磁波（光子） | 最弱 | 最强（需厚铅） | 不偏转 | c |

**偏转方向判定**（磁场中）：
- 正电荷（α）：等效电流方向与运动方向相同 → 用左手定则（四指电流，磁感线穿手心，拇指为力方向）。
- 负电荷（β）：等效电流方向与运动方向相反 → 偏转方向与正电荷相反。
- γ 不带电，不偏转。

### 5.2.3 放射性衰变

- **衰变**：不稳定原子核自发、随机地发射 α、β 或 γ 射线，变为更稳定的核。

#### α 衰变
- 放出 α 粒子（^4_2 He），质量数减 4，质子数减 2。
- 一般式：`^A_Z X → ^(A-4)_(Z-2) Y + ^4_2 He`
- 示例：`^238_92 U → ^234_90 Th + ^4_2 He`

#### β 衰变
- 核内一个中子转为一个质子，放出电子（β⁻）和反中微子。
- 中子 → 质子 + 电子（n → p + e⁻）
- 质量数不变，质子数加 1。
- 一般式：`^A_Z X → ^A_(Z+1) Y + ^0_-1 e`
- 示例：`^14_6 C → ^14_7 N + ^0_-1 e`

#### γ 辐射
- 伴随 α 或 β 衰变，原子核从高能态降至低能态放出光子。
- A、Z 均不变。

### 5.2.4 半衰期

- **定义**：一定量放射性原子核，有一半发生衰变所需的时间。
- 半衰期是固有性质，不受外界影响（温度、压力等）。
- 经过 n 个半衰期，剩余未衰变比例为 (1/2)^n，计数率也按此比例降低。
- **从衰变曲线求半衰期**：在计数率-时间图上，任一点计数率降为一半的时间差即为半衰期。有本底时先减去本底。

#### 应用与半衰期选择

| 应用 | 辐射类型 | 半衰期要求 | 例子 |
|------|----------|------------|------|
| 烟雾报警器 | α | 长（数百年） | 镅-241（432 年） |
| 食品辐照杀菌 | γ | 中等（几年） | 钴-60（5.27 年） |
| 医疗器械灭菌 | γ | 中等 | 钴-60 |
| 厚度控制 | β | 长（维持稳定） | 锶-90（28.8 年） |
| 癌症治疗（外照射） | γ | 适中 | 钴-60 |
| 癌症诊断（示踪剂） | γ | 短（几小时） | 锝-99m（6 小时） |

### 5.2.5 安全防护

#### 危害
- 破坏 DNA → 细胞死亡、突变、癌症。胎儿及快速分裂细胞更敏感。

#### 安全操作
- 用屏蔽容器（铅罐）运输存储。
- 远程操作工具（长柄钳、机械手）。
- 废料深埋地质处置。
- 佩戴剂量计监测累积剂量。

#### 防护三原则
1. **时间**：缩短暴露时间。
2. **距离**：增大距离，辐射强度 ∝ 1/距离²。
3. **屏蔽**：
   - α：纸或手套，防吸入/食入。
   - β：有机玻璃或几 mm 铝（避免重材料产生韧致辐射）。
   - γ：厚铅、混凝土。

## 常见混淆点与记忆提示

- 原子核体积小但质量大。
- α = 氦核，β = 电子，γ = 光子。
- α 衰变：A 减 4，Z 减 2；β 衰变：A 不变，Z 加 1。
- 半衰期不因外界改变，剩余比例 = (1/2)^n。
- 测量放射性务必扣除本底。
- 同位素化学性质相同，但可能放射性不同。
